# ThirtyFoundry public tracker

ThirtyFoundry is a 30-day experiment in shipping narrow, useful tools for overlooked independent businesses. The portfolio hub is configured as `https://thirtyfoundry.vercel.app` and remains a placeholder until deployment.

`roadmap.json` is the source plan; `progress.json` is the automation-owned record of actual work. No secrets belong in this public tracker.

## Status legend

| Status | Meaning |
|---|---|
| `planned` | Selected for the roadmap; work has not begun |
| `building` | At least one build phase is in progress |
| `live` | Tested and published with a usable URL |
| `blocked` | Work stopped on a recorded external dependency |

See [roadmap.md](roadmap.md) for the human-readable plan and `daily-log/` for build decisions.
